## Trigonometric equations

Trigonometric equations can be solved using the algebraic methods, trigonometric identities and values.

#### Example 1

Solve the equation $2\cos\theta - 1 = 0$, $0 \le \theta < 2\pi$.

**Solution:**
When we rearrange the above equation, we get $\cos\theta = \frac{1}{2}$.
For the reference angle $\theta_R$, we have $\cos\theta_R = \frac{1}{2}$ and hence, $\theta_R = \frac{\pi}{3}$. Using the reference angle $\theta_R$, we determine the solution $\theta$ in the interval $[0, 2\pi)$ of the given equation. $\cos\theta = \frac{1}{2}$ suggests that $\cos\theta$ is positive and that means the terminal side of $\theta$ solution to the given equation is either in quadrant I or IV. Hence,
$\theta_1 = \theta_R = \frac{\pi}{3}$
$\theta_2 = 2\pi - \theta_R = 2\pi - \frac{\pi}{3} = \frac{5\pi}{3}$.

#### Example 2

Solve the trigonometric equation $2\sin\theta = -1$.

**Solution:**
Rewrite the above equation in simple form as shown below:
$\sin\theta = -\frac{1}{2}$. The reference angle $\theta_R$ such that $\sin\theta_R = \frac{1}{2}$ is $\theta_R = \frac{\pi}{6}$.

Use the reference angle $\theta_R$ to determine the solutions $\theta_1$ and $\theta_2$ on the interval $[0, 2\pi)$ of the given equation. The equation $\sin\theta = -\frac{1}{2}$ suggests that $\sin\theta$ is negative and that means the terminal side of angle $\theta$ is either in quadrant III or IV as shown in the unit circle Figure 4.34.

#### Example 3

Find all the solutions of the trigonometric equation $\sqrt{3}\sec\theta + 2 = 0$.

**Solution:**
Using the identity $\sec\theta = \frac{1}{\cos\theta}$, we rewrite the equation in the form
$\sqrt{3}\frac{1}{\cos\theta} + 2 = 0 \implies \frac{\sqrt{3}}{\cos\theta} = -2 \implies \cos\theta = -\frac{\sqrt{3}}{2}$.

Find the reference angle $\theta_R$ by solving $\cos\theta_R = \frac{\sqrt{3}}{2}$ for $\theta_R$ acute. Accordingly, $\theta_R = \frac{\pi}{6}$. Using the reference angle $\theta_R$ in the interval $[0, 2\pi)$ of the given equation. $\cos\theta = -\frac{\sqrt{3}}{2}$. This suggests that $\cos\theta$ is negative and that means the terminal side of angle $\theta$ is either in quadrants II or III as shown in Figure 4.35.

or
$\theta_2 = \pi + \theta_R = \pi + \frac{\pi}{6} = \frac{7\pi}{6}$

Use the solutions on the interval $[0, 2\pi)$ to find all solutions by adding multiples of $2\pi$ as follows:
$\theta_1 = \frac{5\pi}{6} + 2n\pi$ and $\theta_2 = \frac{7\pi}{6} + 2n\pi$ where $n$ is an integer.
