Based on the subjects in your list, here is a professional and easy-to-use coding system. This format uses a **3-4 letter prefix** for the subject followed by the **Unit number**, making it perfect for folder naming, file organization, or class schedules.

### Recommended Coding Format: `SubjectCode-U#`

| Subject | Suggested Code | Example (Unit 1) |
| :--- | :--- | :--- |
| **Afan Oromo** | `AFAN` or `AO` | `AFAN-U1` |
| **Amharic** | `AMH` | `AMH-U1` |
| **Biology** | `BIO` | `BIO-U1` |
| **Chemistry** | `CHEM` | `CHEM-U1` |
| **Citizenship** | `CITZ` | `CITZ-U1` |
| **Economics** | `ECON` | `ECON-U1` |
| **English** | `ENG` | `ENG-U1` |
| **Geography** | `GEO` | `GEO-U1` |
| **History** | `HIST` | `HIST-U1` |
| **HPE** | `HPE` | `HPE-U1` |
| **ICT** | `ICT` | `ICT-U1` |
| **Mathematics** | `MATH` | `MATH-U1` |
| **Physics** | `PHYS` | `PHYS-U1` |

### Why use this system?
*   **Sortable:** If you name your files starting with these codes (e.g., `BIO-U1_Notes`), they will automatically group together alphabetically in your computer folders.
*   **Clear:** Anyone looking at the code `PHYS-U3` immediately knows it refers to Physics Unit 3.
*   **Searchable:** You can quickly find all materials for a specific subject just by typing the 3-letter code into your search bar.

**Pro-Tip:** If you are managing multiple grade levels, you can add the grade at the beginning like this: **`G11-BIO-U1`** (Grade 11, Biology, Unit 1).
